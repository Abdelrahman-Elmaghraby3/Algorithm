This repository contains clean, well-documented Java implementations of 
the most essential sorting and searching algorithms — written as a 
learning reference with step-by-step comments and Big O analysis for 
every method.

Sorting Algorithms:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort

Searching Algorithms:
  - Linear Search
  - Binary Search (Iterative)
  - Binary Search (Recursive)

Also includes a full Binary Search Tree (BST) with:
  - Insert / Contains / Remove
  - BFS (Level-Order Traversal)
  - DFS Pre-Order / In-Order / Post-Order

Every algorithm includes:
  - Clear inline comments explaining each step
  - Time and space complexity (Best / Average / Worst)
  - Visual examples in the documentation
